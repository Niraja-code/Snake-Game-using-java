# Snake Game in Java 🐍

## Project Description
A simple console-based Snake Game built using Java and Object-Oriented Programming (OOP). The game includes snake movement, food generation, collision detection, and game-over logic.

---

## Features
- Snake movement in 4 directions
- Random food generation
- Snake grows after eating food
- Self-collision detection
- Boundary collision detection
- Console-based gameplay

---

## Technologies Used
- Java
- OOP Concepts
- LinkedList Data Structure

---

## How to Run

### Compile
javac Game.java

### Run
java Game

---

## Project Structure
Game.java contains:
- Game class (main logic)
- Snake class (snake behavior)
- Board class (grid and food)
- Cell class (grid unit)
- CellType enum (EMPTY, FOOD, SNAKE_NODE)

---

## Game Rules
- Snake moves automatically in a direction
- Eating food increases snake length
- Hitting wall ends game
- Hitting itself ends game
